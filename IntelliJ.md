## 快捷键

| 快捷键              | 作用                 |
| ------------------- | -------------------- |
| ctrl + alt + y      | 刷新，同步           |
| ctrl + alt + 左箭头 | 返回上一个地方       |
| ctrl + alt + 右箭头 | 返回下一个地方       |
| shift + F10         | 运行                 |
| shift + F9          | Debug                |
| ctrl + e            | 最近打开过的文件记录 |
|                     |                      |
|                     |                      |
|                     |                      |
